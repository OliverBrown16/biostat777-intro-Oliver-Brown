## Technical Details

This website was built using **Quarto**. The data analysis and visualizations were created using the **R programming language**, specifically leveraging the `tidyverse` package for data wrangling and `ggplot2` for plotting.

### Key Tools Used:
* **Quarto**: For rendering the website from `.qmd` (Quarto Markdown) files.
* **R**: For all data analysis and plotting.
* **R Packages**: `tidyverse`, `AmesHousing`, `scales`.
* **Deployment**: The site is deployed automatically using **GitHub Actions** and hosted on **GitHub Pages**.
